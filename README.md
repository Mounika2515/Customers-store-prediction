# Customer Segmentation using K-Means Clustering

## Objective
Group customers based on purchasing behavior using K-Means Clustering.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Dataset
Mall Customers Dataset

## Steps Performed
1. Data Loading
2. Feature Selection
3. Elbow Method
4. K-Means Clustering
5. Cluster Visualization

## Results
Successfully segmented customers into 5 clusters based on Annual Income and Spending Score.

## Output
- Elbow Method Graph
- Customer Segmentation Graph
